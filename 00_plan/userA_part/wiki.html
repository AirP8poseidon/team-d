클러스터 사용 방법

  1. 1) SSH로 로그인 노드 접속

    사내 VPN 연결 후 본인 사번 계정으로 접속합니다. 최초 접속 시 발급받은 공개키를 등록하세요.

    $ ssh 사번@gpu-cluster.hpc.corp.local
# 비밀번호 대신 SSH 키 사용 권장
$ ssh-copy-id 사번@gpu-cluster.hpc.corp.local

  2. 2) 환경 설정 (module load / conda activate)

    필요한 모듈을 불러오거나 본인 conda 환경을 활성화합니다.

    $ module avail              # 사용 가능한 모듈 확인
$ module load cuda/12.2 openmpi/4.1
$ conda activate myenv      # 개인 conda 환경 활성화

  3. 3) 작업 제출 (sbatch)

    로그인 노드에서 직접 학습을 돌리지 말고, 반드시 Slurm 배치 스크립트로 제출하세요.

    #!/bin/bash
#SBATCH --job-name=train-resnet
#SBATCH --partition=gpu
#SBATCH --nodes=1
#SBATCH --gres=gpu:a100:2
#SBATCH --cpus-per-task=16
#SBATCH --mem=128G
#SBATCH --time=12:00:00
#SBATCH --output=logs/%x-%j.out

module load cuda/12.2
conda activate myenv

srun python train.py --epochs 50 --batch-size 256
    $ sbatch train.sh          # 위 스크립트를 train.sh로 저장 후 제출
Submitted batch job 184302

  4. 4) 상태 확인 (squeue)

    내 작업의 대기/실행 상태를 확인하고, 필요 시 취소합니다.

    $ squeue --me            # 내 작업만 조회
$ scontrol show job 184302  # 상세 정보
$ scancel 184302         # 작업 취소
